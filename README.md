# unofficial-codex-wiki

`unofficial-codex-wiki` is an unofficial private local mirror of selected OpenAI Codex documentation for offline reading, local search, and AI-agent context ingestion.

This project is not affiliated with, endorsed by, or maintained by OpenAI.

## Current Status

Phase 0 agent customization is complete.

Milestone 1 scaffold and domain types are complete. The repository now has a pnpm workspace, Turborepo config, TypeScript config, package skeletons, CLI skeleton, core domain types, cross-platform path utilities, shared logger, config schema basics, and test setup.

Milestone 2 discovery and safe fetching are complete. The repository can discover Codex Markdown page URLs from `/codex/llms.txt`, fetch raw Markdown into `data/`, reuse cache, and enforce offline mode.

Milestone 3 has not started. Extraction, transformation, generated Markdown, chunking, indexing, validation pipeline behavior, and docs sync behavior are intentionally not implemented yet.

## Private-Use Boundary

This is an unofficial private local mirror of selected OpenAI Codex documentation.
It is intended for local/internal reading and AI-agent context ingestion only.
Do not publish copied OpenAI documentation generated by this project as a public website.

## V1 Direction

V1 will build a deterministic local documentation mirror:

```text
official Codex docs
-> URL discovery from /codex/llms.txt
-> Markdown-first page fetch
-> generated Markdown
-> agent JSONL pages and chunks
-> SQLite FTS5 search
-> CLI read/search/validate
```

V1 must remain CLI-first, private-use-only, and cross-platform for Ubuntu/Linux, Windows, and macOS.
